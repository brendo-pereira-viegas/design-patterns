<section>
   <p>
      O padrão <strong>Prototype</strong> permite criar novos objetos a partir
      de um existente, ou seja, copiando-o, evitando a necessidade de instanciar
      classes diretamente. É muito útil quando há objetos com muitos parâmetros
      ou configurações complexas, promovendo flexibilidade, pois novos objetos
      podem ser gerados em tempo de execução sem depender de subclasses.
   </p>
</section>
