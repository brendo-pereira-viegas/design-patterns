   <h1>Overview</h1>
</section>

<section>
   <p>
      <strong>Published in 1994</strong> in the book <strong>Design Patterns:
      Elements of Reusable Object-Oriented Software</strong>, 
      it is a set of reusable solutions to recurring problems in
      <strong>software development</strong>, applicable across different
      programming languages. These are high-level concepts used to address
      <strong>specific situations</strong>.
   </p>
</section>

<section>
   <ul>
      <li>
         Creationals: These patterns focus on how objects are created, abstracting the process to make the code more flexible and reusable, while hiding the implementation and easing adaptation to changes. They centralize object creation logic to improve consistency. They also support lazy initialization, reducing unnecessary resource usage..
      </li>
   </ul>
</section>
