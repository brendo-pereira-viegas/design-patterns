<section align="center">
   <h1>Overview</h1>
</section>

<section>
   <p>
      <strong>Published in 1994</strong> in the book <strong>Design Patterns:
      Elements of Reusable Object-Oriented Software</strong>, 
      it is a set of reusable solutions to recurring problems in
      <strong>software development</strong>, applicable across different
      programming languages. These are high-level concepts used to address
      <strong>specific situations</strong>.
   </p>
</section>

<section>
   <ul>
      <li>
         <strong>Creationals</strong>: These patterns focus on
         <strong>how objects are created</strong>, abstracting <br>
         the process to make the code more <strong>flexible and
         reusable</strong>, while hiding the implementation and easing
         adaptation to changes.
      </li>
   </ul>
</section>
